# Solution C

Ask for an exception extension so the partner workflow is not broken next week. Security should review the exploit signal and confirm whether it applies. If the signal is not clearly applicable, ship with monitoring and plan to patch after partner coordination.

Audit should receive the extension ticket, the product owner's impact statement, and security's notes. The old scanner baseline should be refreshed when there is time, but the partner contract makes the immediate release important.

If security says the exploit is relevant, block release. Otherwise accept the short-term risk and track the patch date.
