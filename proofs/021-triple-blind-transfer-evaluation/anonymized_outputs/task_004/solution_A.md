# Solution A

Try the cataloged backup first because it is the intended restore source. Before starting, compare the catalog entry with the object-store inventory and ask the backup owner to explain the missing segment. If the segment is required, do not sign the rehearsal as complete.

The replica can be kept as a fallback, but security must renew or approve the account before it is used. The team should start before the freeze and record each command, source identifier, and validation result.

If the backup restores and application checks pass, sign the rehearsal. If the missing segment blocks restore, mark the rehearsal incomplete and schedule a follow-up after freeze approval.
