# Solution C

Move some onboarding agents to enterprise cases because the frontline tickets show aging there. Refresh the dashboard when possible, but do not wait if enterprise tickets are breaching.

The operations lead should tell workforce which agents are needed and ask product whether the fix has reduced incoming volume. VIP accounts should be handled first after the list is updated.

If the moved agents cannot resolve the tickets, bring them back and ask for specialist help. The decision should be revisited after the dashboard refresh.
