# Problem Statement

Long-horizon agent systems preserve operational continuity by saving memory, condensing history, replaying traces, resuming work, and reconstructing state after compression or restart. These mechanisms can keep useful beliefs available over time, but they also raise a distinct authority question: when a remembered belief survives, what makes it operationally authoritative?

Proof 006 asks:

When long-horizon systems promote, summarize, replay, or recover memory state, how is operational authority assigned, preserved, superseded, contradicted, or degraded over time?

The secondary question is:

Which continuity gaps can cause stale, compressed, inferred, or partially reconstructed memory to become operationally authoritative?

## Scope

This proof is limited to public documentation, public replay or trajectory artifacts when available, public issue or discussion records when later collected, and public metadata about memory systems, memory recall, memory promotion, memory persistence, summaries, compaction, condensation, replay, events, recovery, and restart paths.

Initial targets are OpenClaw and OpenHands. The proof focuses on authority propagation surfaces, not private implementation internals.

## Out Of Scope

- mechanistic interpretability
- AGI claims
- consciousness claims
- black-box-solved claims
- substrate disclosure
- vulnerability research
- target-system attack framing
- Belief Ledger internals
- Inside Voice internals
- harmonic traceback implementation details
- private prompts, private traces, or hidden chain-of-thought

## Evidence Standard

This proof distinguishes observed public mechanisms from inferred survivability implications and unresolved authority questions. A memory, summary, recall, or replay surface is not treated as authority-preserving unless it exposes enough lineage to recover source authority, supersession state, temporal validity, stale-belief status, contradiction status, omitted ranges, memory promotion provenance, and replay provenance.

The current phase is evidence collection. All dimensions, taxonomy categories, and matrix assessments are preliminary and subject to source-backed revision.
