# Ambiguity Survivability Taxonomy

Each category below is an initial taxonomy, subject to evidence validation. These are possible ambiguity-preservation limitations, not conclusions about either target system.

## Contradiction flattening

Compression, condensation, memory promotion, or replay views may preserve a readable narrative while dropping explicit conflicts. This category applies only when evidence shows that conflicting source material is not recoverable or not linked after transformation.

## Uncertainty collapse

Uncertainty markers such as unknowns, caveats, confidence limits, and unresolved questions may be weakened or lost. This category remains conservative until evidence shows whether uncertainty is preserved, summarized, or omitted.

## Replay ambiguity loss

Replay may show what happened while implying more causal certainty than the artifact supports. This category applies when visible replay lacks enough provenance to reconstruct competing interpretations, omitted ranges, or missing state.

## Stale assumption persistence

Old assumptions may survive in summaries or memory after they become contradicted, superseded, or temporally invalid. This category requires evidence of validity-boundary loss before it can support a target-specific claim.

## Rejected-alternative erasure

Alternatives considered but rejected may disappear from condensed context, making replay look more linear than the original operational state. This category is a hypothesis until specific artifacts show whether rejected alternatives are retained.

## Authority ambiguity loss

Source authority can be blurred when user instructions, assistant inferences, tool outputs, memory recalls, and documentation references are merged into a summary. Evidence must show which authority metadata is present or absent before stronger claims are made.

## Temporal contradiction collapse

Before/after conflicts, corrections, retries, restarts, and supersessions can be compressed into timeless prose. This category applies when temporal ordering or validity windows are not reconstructable from public artifacts.

## Memory contradiction drift

Memory can preserve or reintroduce claims without preserving the contradictions, caveats, source authority, or time bounds attached to those claims. This category remains unresolved until memory write, recall, and provenance surfaces are examined.

## Summary simplification drift

Summaries may become operationally useful but evidentially thinner over repeated compression or reconstruction. This category asks whether summary lineage preserves omitted ranges, source refs, contradiction refs, and uncertainty markers.

## Operational truth compression loss

Operational truth can degrade when compressed, replayed, or promoted context loses the difference between observed fact, inference, uncertainty, stale assumption, and unresolved contradiction. This category is not a defect claim; it defines the proof's highest-level audit risk.
