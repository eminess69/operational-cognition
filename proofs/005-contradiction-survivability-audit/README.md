# Contradiction Survivability Audit for Long-Horizon Agent Systems

## Objective

Evaluate whether contradictions, uncertainty, stale assumptions, rejected alternatives, source-authority ambiguity, and temporal conflicts remain reconstructable when long-horizon agent context is compressed, summarized, replayed, reconstructed, or promoted into memory.

## Target Systems

- OpenClaw
- OpenHands

## Primary Question

When long-horizon context is compressed, summarized, replayed, reconstructed, or promoted into memory, do contradictions and uncertainty remain reconstructable?

## Secondary Question

Which operational mechanisms risk flattening ambiguity into simplified replay narratives?
