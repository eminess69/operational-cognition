# Replayable Continuity Audit of Long-Horizon Agent Failure

## Objective

Investigate whether long-horizon agent failures can be reconstructed through replayable public evidence, continuity drift analysis, and operational lineage summaries better than ordinary issue-level debugging.

## Target Systems

- OpenClaw
- OpenHands

## Status

Planning scaffold only. This proof is not complete and does not yet contain gathered evidence, validated findings, or final conclusions.

## Expected Outputs

- problem statement
- evidence plan
- initial failure taxonomy
- replayability plan
- mitigation doctrine
- public lineage summary
- deterministic proof manifest

## Public/Private Boundary

This proof pack may publish public evidence references, taxonomy, replay plans, mitigation doctrine, and sanitized lineage summaries.

It must not publish Belief Ledger internals, Inside Voice implementation details, hidden chain-of-thought, private scoring, harmonic traceback implementation, substrate state, secrets, or local-only private paths.

## Current Limitation

The Inside Voice MCP bridge used for planning is active, but the adapter status is `placeholder`. The MCP consultation response is advisory planning context only and does not represent live deep substrate cognition.
