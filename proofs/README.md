# Proof Packs

This directory contains public proof packs for Operational Cognition.

Each proof pack is a bounded investigation scaffold or evidence-backed analysis package. A proof pack must follow `docs/PROOF_STANDARD.md`, preserve the public/private boundary, and include a valid `proof_manifest.json`.

## Current Proofs

- `001-agent-continuity-audit`: Replayable Continuity Audit of Long-Horizon Agent Failure
- `002-replayability-gap-audit`: Replayability Gap Audit for Long-Horizon Agent Systems
- `003-continuity-compression-audit`: Continuity Compression Audit for Long-Horizon Agent Systems
- `004-operational-failure-reconstruction`: Operational Failure Reconstruction Audit for Long-Horizon Agent Systems

## Status Language

Proof status must be one of:

- `planning`
- `evidence_collection`
- `analysis`
- `complete`
- `superseded`

Planning proof packs are scaffolds. They do not claim completed investigation results.
